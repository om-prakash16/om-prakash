# Om Prakash Kumar — Portfolio

Professional portfolio of **Om Prakash Kumar**, Python Developer & Process Associate. Specialist in automation, data pipelines, and high-performance web engineering.

Live at: **[ommee.in](https://ommee.in)**

## 🛠️ Tech Stack

- **Core:** Python (Pandas, NumPy, Automation)
- **Frontend:** Next.js 15, TypeScript, Tailwind CSS, Framer Motion
- **Creative:** Three.js / @react-three/fiber
- **Operations:** Bill.com, RetailBackbone, Microsoft Excel
- **Background:** Technical Writing @ GeeksforGeeks

## 🚀 Key Features

- **Cinematic UI:** Awwwards-level aesthetic with custom cursors, holographic effects, and glassmorphism.
- **Smooth UX:** Lenis inertia scrolling with scroll-progress tracking and active section indicators.
- **Performance:** Optimized for speed and responsiveness across all devices (Mobile/Desktop).
- **SEO Optimized:** Full OpenGraph, Twitter cards, and JSON-LD structured data for professional indexing.

## 📁 Structure

- `/src/components/sections`: All core portfolio sections (Hero, About, Projects, etc.)
- `/src/components/canvas`: 3D R3F components and models
- `/src/components/ui`: Reusable high-level UI components (Navbar, SmoothScroll)
- `/src/app/globals.css`: Fluid typography and design system tokens
